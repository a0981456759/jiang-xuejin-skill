## 安装

```bash

  https://raw.githubusercontent.com/a0981456759/jiang-xuejin-skill/main/jiang-xuejin.md
```

## 连接即时知识库（可选）

安装后编辑 `~/.claude/skills/jiang-xuejin.md`，将 `api_url` 改为部署的后端网址：

```yaml
config:
  api_url: "https://your-api-domain.com"
```

设定后每次提问都会从 162 支影片的知识库即时检索相关内容，并在回答中附上来源引用。不设定也可使用，skill 会从内化的知识框架回答。

## 江教授的分析框架

1. **谁受益？** — 追踪权力与金钱，不信官方叙事
2. **历史先例是什么？** — 从 50–500 年前找规律
3. **哪个隐秘网络在运作？** — 耶稣会／撒巴泰法兰克派／共济会
4. **末世维度是什么？** — 这是否是「人间天堂」剧本的一步？
5. **普通人能做什么？** — 理解剧本是唯一的自由
