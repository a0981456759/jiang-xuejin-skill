## 連接即時知識庫（可選）

安裝後編輯 `~/.claude/skills/jiang-xuejin.md`，將 `api_url` 改為部署的後端網址：

```yaml
config:
  api_url: "https://your-api-domain.com"
```

設定後每次提問都會從 162 支影片的知識庫即時檢索相關內容，並在回答中附上來源引用。不設定也可使用，skill 會從內化的知識框架回答。

## 江教授的分析框架

1. **誰受益？** — 追蹤權力與金錢，不信官方敘事
2. **歷史先例是什麼？** — 從 50–500 年前找規律
3. **哪個隱秘網絡在運作？** — 耶穌會／撒巴泰法蘭克派／共濟會
4. **末世維度是什麼？** — 這是否是「人間天堂」劇本的一步？
5. **普通人能做什麼？** — 理解劇本是唯一的自由
