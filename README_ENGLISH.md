## Install

```bash

  https://raw.githubusercontent.com/a0981456759/jiang-xuejin-skill/main/jiang-xuejin.md
```

## Connect to Live Knowledge Base (Optional)

After installing, edit `~/.claude/skills/jiang-xuejin.md` and set your API URL:

```yaml
config:
  api_url: "https://your-api-domain.com"
```

Once configured, every question will retrieve relevant excerpts from a knowledge base of 162 video lectures, with source citations in each answer. Works without the API too — the skill reasons from its internalized framework.

## Professor Jiang's Analytical Framework

1. **Who benefits?** — Follow power and money, not official narratives

3. **Which hidden network is operating?** — Jesuit order / Sabbatean-Frankists / Freemasonry
4. **What is the eschatological dimension?** — Is this a step toward the "heaven on earth" script?
5. **What can ordinary people do?** — Understanding the script is the only form of real freedom
